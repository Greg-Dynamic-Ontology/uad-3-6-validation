# ADR-0014 — RDF Representation of XML Schema Components

## Schema Packaging Principle
The UAD 3.6 XML Schema is published in both Combined and Individual forms. These are alternative physical organizations of the same logical schema.

The generator shall support both distributions. Schema imports and includes shall be resolved 
before RDF generation so that generated RDF artifacts depend on the logical schema components 
rather than on their physical file organization.

| XSD component  | Notes                                                                                                                                      | RDF representation                                                                                                                             |
|----------------|--------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| annotation     | Documentation container.                                                                                                                   | RDF annotations (container only; does not become a domain resource)                                                                            |
| any            | Not used by the UAD RDF projection.                                                                                                        | ignore                                                                                                                                         |
| anyAttribute   | Not used by the UAD RDF projection.                                                                                                        | ignore                                                                                                                                         |
| attribute      | Named literal-valued property.                                                                                                             | `owl:DatatypeProperty`                                                                                                                         |
| attributeGroup | Reusable collection of attributes.                                                                                                         | No RDF resource is created. During XSD→RDF processing, each referenced attribute is projected as if it were declared locally.                  |
| choice         | Alternative content model. During projection, process only the branch present in the instance.                                             | **No direct representation** (projection rule only)                                                                                            |
| complexType    | Named structured type.                                                                                                                     | `owl:Class`                                                                                                                                    |
| documentation  | Human-readable documentation.                                                                                                              | `rdfs:comment`, `skos:definition`, or other annotation property                                                                                |
| element        | Depends on declaration and usage. See "Elements table" below                                                                               | See Elements table below                                                                                                                       |
| enumeration    | Controlled vocabulary member.                                                                                                              | `skos:Concept`                                                                                                                                 |
| extension      | In MISMO, commonly extends a Base simple type containing the scalar or enumeration value so attributes can be added through simpleContent. | Inherit the base datatype or enumeration mapping, then add the declared attributes. Do not infer rdfs:subClassOf solely from this XSD pattern. |
| fractionDigits | Numeric datatype facet.                                                                                                                    | SHACL constraint                                                                                                                               |
| import         | Reference to external schema of different namespace.                                                                                       | Resolve imported schema components before generation. Emit `owl:imports` only when generating separate ontology modules.                       |
| include        | Reference to external schema of same namespace.                                                                                            | No direct RDF representation. Resolve included schema components before generation; process included declarations as though declared locally.  | 
| maxInclusive   | Datatype facet.                                                                                                                            | SHACL constraint                                                                                                                               |
| maxLength      | Datatype facet.                                                                                                                            | SHACL constraint                                                                                                                               |
| minInclusive   | Datatype facet.                                                                                                                            | SHACL constraint                                                                                                                               |
| pattern        | Datatype facet.                                                                                                                            | SHACL constraint                                                                                                                               |
| restriction    | Type/datatype restriction.                                                                                                                 | SHACL constraint (possibly OWL in limited cases)                                                                                               |
| sequence       | Process each child declaration; XML order carries no UAD business semantics.                                                               | Order has no semantic meaning                                                                                                                  |
| simpleContent  | Literal content with attributes.                                                                                                           | Project the value using the referenced XSD datatype (e.g., xsd:string, xsd:decimal, xsd:date).                                                 |
| simpleType     | Scalar datatype or enumeration definition.                                                                                                 | `rdfs:Datatype` **or** `skos:ConceptScheme` (depending on definition)                                                                          |
| union          | Defines a simple type whose value may conform to any listed member datatype.                                                               | Connect the named simple type to each datatype listed in memberTypes. No separate resource for the union element is required.                  |

| Elements table                  | Notes | RDF representation             |
|---------------------------------|-------|--------------------------------|
| Element typed by a complex type |       | object/resource                |
| Element typed by a simple type  |       | literal property.              |
| Element using ref               |       | use the referenced declaration |
| Repeating element               |       | repeated property assertions   |
| Root element                    |       | root resource of the RDF graph |
